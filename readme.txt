Cerakote Fetcher v1.0
Written by Dale Stevens
8/31/2023

Fetches all Cerakote colors by SKU, Name and Image URL, will also download image swatches of each color into the /swatch_images folder.

Usage:

Start -> Powershell (Admin) cd to Cerakote-Fetcher.ps1
type .\Cerakote-Fetcher.ps1 and hit Enter.

Enjoy
